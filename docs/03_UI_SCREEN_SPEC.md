# 03｜UI 页面规格

## 总体视觉方向

关键词：

- 自然系数字艺术
- 高级留白
- 半透明材质
- 柔和雾感
- 低饱和植物色
- 细腻颗粒
- 微弱呼吸动效
- 非卡通、非儿童化

避免：

- 大面积高饱和绿色
- 卡通农场风
- 游戏化金币、等级、生命值
- 医疗蓝色 Dashboard
- 过多圆角卡片堆叠
- 过度玻璃拟态导致可读性差

## 建议画布

优先按移动端设计：

- 基准宽度：390px
- 基准高度：844px
- 同时兼容 360px 至 430px
- 桌面端最大内容宽度：480px 或 520px
- 桌面背景可以延展为氛围环境，不把手机页面强行铺满

## 色彩建议

基础色：

- 背景米白：`#F3F1EA`
- 夜间灰蓝：`#20282A`
- 雾绿：`#A8B5A4`
- 深苔绿：`#35443B`
- 枝干褐：`#796B5B`
- 露珠蓝灰：`#9BAFB4`
- 微光金：`#D9C89B`

颜色必须通过 CSS 变量统一管理。

## 字体方向

中文：优先系统字体或清晰的人文黑体。

标题可以稍带文学感，但正文必须清晰，不使用难阅读的装饰字体。

建议层级：

- 超大标题：32–40px
- 页面标题：24–28px
- 正文：15–17px
- 辅助说明：12–14px

## 页面清单

### A01 启动/欢迎页

视觉中心：一株被雾遮住的植物轮廓。

组件：

- Logo
- 主标题
- 两行副文案
- 主按钮
- 次按钮

### A02 问卷页

组件：

- 顶部返回
- 进度线
- 问题文本
- 3 至 4 个答案卡片
- 植物或光点的小型响应区域

### A03 植物分配结果页

组件：

- 全屏植物主视觉
- 植物名称
- 特质短句
- 匹配说明
- 进入按钮

### B01 花园首页

布局：

- 顶部 15%：日期、问候、设置入口
- 中部 55% 至 60%：植物主视觉
- 下部：植物回应、记录按钮
- 最底部：四项导航

### B02 快速情绪记录

可以使用底部抽屉或全屏页面。

组件：

- 情绪天气选择
- 强度滑动条
- “继续写一点”入口
- 快速提交按钮

### B03 完整记录页

组件：

- 情绪标签
- 文字输入框
- 触发因素
- 身体感受
- 提交按钮

### B04 植物回应页

组件：

- 植物状态动画
- 一句回应
- “放进土里”或“留在叶片上”的仪式按钮
- 返回花园

### C01 情绪日历

组件：

- 月份切换
- 月视图
- 图例
- 选中日期详情卡片

### C02 月度报告

组件：

- 月度植物标本主图
- 三项统计
- 一段总结
- 保存图片按钮占位

### D01 共鸣森林

布局：

- 顶部标题与简介
- 瀑布流或纵向内容卡片
- 发布按钮
- 每条内容只展示匿名植物头像、时间、正文、共鸣按钮

### D02 发布树洞

组件：

- 文本输入
- 情绪天气
- 是否匿名说明
- 发布按钮

### E01 我的

组件：

- 植物档案卡
- 偏好设置
- 隐私
- 数据
- 可穿戴设备占位入口
- 重置按钮

## 动效原则

- 所有动效速度偏慢
- 避免弹跳和夸张缩放
- 页面切换 250–450ms
- 植物呼吸循环 5–9 秒
- 叶片摆动幅度极小
- 雾和光点可使用 CSS 或轻量 Canvas
- 支持 `prefers-reduced-motion`
