# 04｜视觉素材清单

## 先说结论

第一版不需要几十张完整页面设计图。最重要的是准备一套统一风格的“可拆分素材”，让 Codex 用网页布局、CSS 和动效把它们组合起来。

不要把整张 UI 页面直接生成为一张图片后放进网页。那样会导致文字不能编辑、无法响应式适配、按钮不能真正交互。

## A. 必须准备的核心素材

### 1. 主植物状态图

建议先选一株植物做 MVP，例如银叶蕨、睡莲或虚构植物。

至少需要 8 个状态：

1. 初始朦胧状态
2. 平静舒展
3. 开心微光
4. 悲伤露珠
5. 焦虑轻颤/叶片收拢
6. 愤怒深色脉络
7. 疲惫低垂
8. 复杂情绪双色光影

规格建议：

- PNG 或 WebP
- 透明背景
- 2048×2048 或更高
- 植物位置和大小尽量一致
- 光源方向一致
- 不包含文字、按钮和 UI

### 2. 植物生长阶段图

至少 5 个阶段：

1. 种子
2. 发芽
3. 小苗
4. 成长期
5. 成熟形态

这些阶段用于首次分配、长期记录和月度成长。

### 3. 环境氛围叠加层

需要透明背景素材：

- 薄雾
- 小雨
- 露珠
- 柔和光斑
- 漂浮颗粒
- 水面波纹
- 微风线条
- 深色情绪纹理

要求：

- 可以覆盖在植物图之上
- 边缘自然
- 不要出现完整背景
- 最好能无缝叠加

### 4. 背景图

准备 4 至 6 张即可：

- 清晨花园
- 阴天花园
- 夜间花园
- 雨天花园
- 雾中温室
- 共鸣森林

要求：

- 竖屏 9:16
- 中央区域留空，方便放植物
- 不出现文字和 UI
- 对比度不能过高

### 5. 情绪天气图标

需要 8 个统一风格图标：

- 晴朗
- 阴天
- 小雨
- 雷雨
- 雾
- 风
- 闷热
- 说不清

建议：

- SVG 优先
- 线性或半透明绘制
- 不使用儿童卡通表情

### 6. 匿名植物头像

共鸣森林使用 12 至 20 个小型植物头像。

要求：

- 1:1
- 简洁轮廓
- 同一画风
- 不像真人头像
- 不同叶型和颜色即可

## B. 可后补的增强素材

- 植物标本卡背景
- 根系纹理
- 种子胶囊
- 漂流瓶
- 月度报告装饰
- 空状态插画
- 设置页小图标
- 品牌 Logo
- PWA 图标
- 启动画面

## C. 不建议生成的素材

不要让图像模型生成：

- 带中文文字的完整页面
- 导航栏截图
- 按钮文字
- 数据图表
- 输入框
- 日历数字
- 多个页面拼在一起的 UI 展示图

这些应由真实前端代码实现。

## D. 素材命名规范

示例：

```text
assets/
  plants/
    fern-base-calm.webp
    fern-happy-glow.webp
    fern-sad-dew.webp
    fern-anxious-folded.webp
    fern-angry-veins.webp
    fern-tired-droop.webp
  growth/
    fern-seed.webp
    fern-sprout.webp
    fern-young.webp
    fern-growing.webp
    fern-mature.webp
  overlays/
    fog-soft.webp
    rain-light.webp
    dew-drops.webp
    particles-gold.webp
  backgrounds/
    garden-morning.webp
    garden-rain.webp
    garden-night.webp
    forest-community.webp
  icons/
    weather-sunny.svg
    weather-cloudy.svg
```

## E. 第一批最少素材包

为了尽快开始开发，第一批只准备：

- 1 株主植物的 6 个情绪状态
- 3 张背景
- 8 个天气图标
- 8 个匿名植物头像
- 4 个氛围叠加层
- 1 个 Logo

总计大约 30 个素材，已经足够做出完整第一版。
