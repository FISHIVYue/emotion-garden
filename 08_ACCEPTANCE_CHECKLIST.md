# 08｜第一版验收清单

## 功能

- [ ] 首次进入显示欢迎页
- [ ] 可以完成植物匹配问卷
- [ ] 问卷完成后分配植物
- [ ] 再次进入不会重复问卷
- [ ] 首页显示植物和当前状态
- [ ] 可以新增情绪记录
- [ ] 记录后植物状态发生变化
- [ ] 情绪记录保存在本地
- [ ] 日历可以显示有记录的日期
- [ ] 可以查看单日记录
- [ ] 森林显示模拟树洞内容
- [ ] 用户可以发布一条本地树洞
- [ ] 可以发送共鸣回应
- [ ] 可以重置全部本地数据
- [ ] 可穿戴设备入口明确标注为后续功能

## 视觉

- [ ] 首页植物是最强视觉中心
- [ ] 页面没有大量数据卡片堆叠
- [ ] 色彩低饱和且统一
- [ ] 所有素材风格统一
- [ ] 植物负面状态不呈现死亡或病害
- [ ] 页面有足够留白
- [ ] 字体在手机上清晰可读
- [ ] 底部导航不遮挡内容

## 动效

- [ ] 植物有轻微呼吸动效
- [ ] 页面切换不突兀
- [ ] 情绪提交后状态切换自然

- [ ] 构建命令可以成功执行
- [ ] 手机端无横向滚动条

- [ ] 主要组件已拆分
