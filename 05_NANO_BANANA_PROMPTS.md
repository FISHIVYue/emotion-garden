# 05｜Nano Banana 素材生成提示词

## 使用原则

每一次生成只做一种素材，不要同时要求生成完整 App 页面。

先确定一张“风格母图”，之后所有提示词都强调：

- 保持同一植物品种
- 保持同一构图
- 保持同一光源
- 保持同一材质
- 保持同一画风
- 不出现文字和 UI

## 1. 主植物基础图

```text
生成一株用于高端情绪记录应用的银叶蕨主视觉素材。植物位于画面中央，完整可见，半透明细腻叶片，轻微雾感，自然数字艺术风格，低饱和灰绿色，柔和侧光，安静、克制、精致，不要卡通，不要花盆，不要人物，不要文字，不要按钮，不要界面。透明背景，正方形构图，植物边缘清晰，适合网页叠加使用。
```

## 2. 平静状态

```text
保持参考图中完全相同的银叶蕨品种、构图、视角、大小、材质和光源。把植物表现为平静舒展状态，叶片自然打开，呼吸感轻微，颜色柔和稳定，少量细小露光。不要改变植物结构，不要背景，不要文字，不要 UI，透明背景。
```

## 3. 悲伤状态

```text
保持参考植物的品种、构图、视角和大小完全一致。表现悲伤但不是枯萎：叶片略微向下，叶尖有少量透明露珠，色调更冷，整体仍然有生命力和美感。禁止出现腐烂、死亡、病害。透明背景，不要文字和 UI。
```

## 4. 焦虑状态

```text
保持参考植物完全一致。表现轻度紧绷和焦虑：部分叶片略微收拢，叶缘有非常细小的振动感线索，叶脉轻微明暗变化，但植物仍健康。低饱和、克制、精致。透明背景，不要文字和 UI。
```

## 5. 开心状态

```text
保持参考植物完全一致。表现温和的开心和轻盈，不要夸张：叶片更加舒展，嫩芽出现，植物周围有少量柔和金色微光，不能像游戏升级特效。透明背景，不要文字和 UI。
```

## 6. 疲惫状态

```text
保持参考植物完全一致。表现疲惫与休息：叶片轻微低垂，色彩略暗，像进入安全的休眠状态，而不是生病或枯萎。植物底部有非常微弱的暖光，表达正在恢复。透明背景，不要文字和 UI。
```

## 7. 环境背景

```text
生成一个用于移动端情绪花园应用的竖屏背景，9:16。安静的现代温室与自然花园融合，低饱和米白、雾绿和灰蓝色，轻微颗粒，柔和晨光，中央区域保持干净和留白，用于放置植物主视觉。不要人物，不要文字，不要按钮，不要完整植物主体，不要卡通。
```

## 8. 雾叠加层

```text
生成用于网页叠加的柔和薄雾素材，半透明、边缘自然、低对比度、无固定场景、无文字、无植物、透明背景，适合覆盖在移动端植物主视觉上。
```

## 9. 匿名植物头像

```text
生成一组高端自然系数字艺术植物头像，1:1，共 12 个，每个是一种不同的小型植物轮廓，低饱和、半透明、细腻、安静、非卡通，背景极简，不出现人脸、文字、图标或边框，整体风格统一。
```

## 10. 风格统一修正提示

```text
请把当前素材调整为与参考图完全统一的视觉系统：相同的低饱和色彩、半透明植物材质、柔和侧光、轻微雾感、精细叶片纹理和克制的高级感。保持原有主体结构与构图，不增加文字、按钮、界面或新物体。
```
